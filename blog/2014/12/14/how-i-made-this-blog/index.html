How I Made This Blog

I wanted a clean and simple way of blogging and wanted to have a clear link between my blog and my github. I started using Tumblr to begin with but I wanted to be able to put code on the blog and have it format correctly which Tumblr didn’t make easily possible.

I settled on Octopress which achieved my aims. I did the following steps:

1.GitHub Setup to get your GitHub ready. Be careful and ensure that your repo is formatted as username.github.io. It will take a few minutes to install. 2.Octopress Install to complete the setup. 3.Finally, I selected and installed my theme, LeanPress. Take a look at Octopress Themes to select yours.

I am also using the markdown cheat sheet to help me get to grips with markdown. I have to say I’m starting to really favour it and it makes it much easier to be explicit about your formatting.

In the future, I would like to design my own blog and not use a theme. For now, I have to focus on Makers.
